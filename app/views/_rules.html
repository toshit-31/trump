Description

Trump is a little variation of the popular american-origin card game, Spades. The only difference it holds with its original version is that here the player chooses a dominating (trump) suit based on the initial cards that he receives for making the call and the game proceedes in the same way as its original version.

Rules

  1. The server decided who will call (caller) the trump suit.
  2. Once the caller decides the trump suit for the entire round, other players are informed.
  3. The player next in turn to the caller will start by playing his/her card.
  4. Now the rest of the players have to play a card of same suit if they have.
  5. If other players don't have a card of the same suit played by the first player in that set he/she can play a card of any suit.
  6. If the set contains trump card then the preference if comparing ranks is given to trump card.
  7. Player playing card with highest rank within a set and the suit same as the first card played in the set wins the set if the set doesn't contain a trump suit.
  8. After the last set is played, the team of two players sitting opposite to each other wins who was more sets collected combined.
